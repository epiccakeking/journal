# Journal

A simple journal application using GTK.

## Installation

From PyPI:

```
pip install epiccakeking-journal
```

From source:

```
git clone https://github.com/epiccakeking/journal.git
pip install ./journal
```

Run without installing:

```
git clone https://github.com/epiccakeking/journal.git
cd journal
python3 -m epiccakeking_journal
```
